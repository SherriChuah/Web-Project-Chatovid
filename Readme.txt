Project Description

- First Year Team Project where we came out with the idea of Chatting During Covid hence Chatovid
- Project consists of an API that updates the daily covid cases in Manchester, a chatting service, a static site to suggest things to do in this unprecedented times and more
- The project uses Node.js as the backend and bootstrap as the frontend
